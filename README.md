# 🧠 ForoHub API

API REST desarrollada con **Spring Boot** para la gestión de tópicos de un foro.  
Este proyecto permite crear, consultar, actualizar y eliminar tópicos utilizando una arquitectura REST conectada a **PostgreSQL**.

Proyecto desarrollado como parte del challenge **ForoHub - Backend**.

---

# 🚀 Tecnologías utilizadas

- Java 21
- Spring Boot
- Spring Data JPA
- PostgreSQL
- Maven
- Jakarta Validation
- Lombok
- REST API
- IntelliJ IDEA
- Postman
